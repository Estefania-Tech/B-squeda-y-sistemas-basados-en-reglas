# Objetivo de la actividad
 - recibir un origen y un destino
 - evaluar las multiples rutas posibles
 - seleccionar la mejor opción en tiempo

El desarrollo del sistema esta basado en representar los conocimientos mediante reglas logicas,
en un motor de inferencia lo que me permite estudiar las diferentes rutas, y se difinio un grafo como modelo
de transporte.

# Grafo
 - Esta definido por nodos que son las paradas, unos Aristas que son las conexiones entre nodos/paradas
 - y un peso determinado como el tiempo, costo, y distancia
